# Changelog

All notable changes to SoWork News will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-17

### Added

#### Core Features
- **Contextual News Reader** — AI-powered news aggregation platform
- **News Clustering** — Automatic grouping of similar articles from multiple sources
- **Turning Point Detection** — AI identifies critical moments that define news stories
- **Perspective Suggestions** — AI-powered discussion angles to avoid echo chambers
- **Real-time Timeline** — Visual timeline of news stories as they break
- **Multi-source Aggregation** — Support for 50,000+ English-language news sources

#### User Features
- **Email/Password Authentication** — Secure sign-up and login system
- **OAuth Support** — Google and GitHub single-sign-on (coming soon)
- **Multi-language Support** — English, Chinese, Japanese support
- **Social Sharing** — OG image optimization for social media sharing
- **Responsive Design** — Works on desktop, tablet, and mobile

#### Technical
- **TypeScript** — Full TypeScript codebase with type safety
- **React 18** — Modern React with hooks
- **Node.js Backend** — Express/Hono HTTP server
- **PostgreSQL** — Relational database with Drizzle ORM
- **OpenAI Integration** — For AI news analysis
- **Perplexity Integration** — For supplemental search

#### Documentation
- Comprehensive README with quick start guide
- Contributing guidelines
- Security policy
- Environment variables template
- API documentation
- Market analysis and globalization strategy
- English localization checklist

### Technical Stack

**Frontend**
- React 18 + TypeScript
- Vite (build tool)
- TailwindCSS (styling)
- Custom UI components

**Backend**
- Node.js + TypeScript
- PostgreSQL database
- Drizzle ORM
- OpenAI API
- Perplexity API

**DevOps**
- pnpm (package manager)
- GitHub Actions (CI/CD ready)
- Docker (deployment ready)

### Known Limitations

- Mobile app not yet available (planned for Q3 2026)
- Email digest feature coming soon
- Advanced recommendation engine under development
- Slack integration planned
